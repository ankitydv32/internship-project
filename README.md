# 🔐 Authentication System (Internship Project)

## 📌 Project Overview

This is a full-stack user authentication system built using **Node.js, Express, MongoDB, and React**.
It allows users to register, login, verify email, and reset passwords securely.

---

## 🚀 Features

* User Registration & Login
* JWT Authentication
* Email Verification
* Forgot Password & Reset Password
* Protected Routes (Dashboard access)
* Admin Panel (optional)

---

## 🛠️ Tech Stack

### 🔹 Backend:

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT (Authentication)
* Bcrypt (Password hashing)

### 🔹 Frontend:

* React.js
* Axios
* React Router

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
cd internship-project
```

---

### 2️⃣ Setup Backend

```
cd backend
npm install
npm start
```

---

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm start
```

---

## 🔐 Environment Variables (.env)

Create a `.env` file in the backend folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
